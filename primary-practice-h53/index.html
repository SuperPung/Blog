兔子繁殖问题

发表于 2021-05-01 分类于 技术 阅读次数： 本文字数： 412 阅读时长 ≈ 1 分钟

Primary Practice h53

1 对兔子出生以后经过 180 天可以生出一窝（2 对）兔子，以后每隔 90 天繁殖一次生出一窝（2 对）兔子

每对兔子的寿命是 700 天

@param startCount 第 0 天 开始的时候初始的兔子对数

@param days 经过的天数

@return 目前系统中存活的兔子的对数
