第一个 Java 小游戏（改错）

Posted on 2021-03-22 Views:
Symbols count in article: 8.9k Reading time ≈ 8 mins.

OOP h07

请确认已经完成 h06。

本作业给出了一部分代码，但是这些代码实现有错误，要么无法编译通过，要么存在错误

不许修改现有类的名字，不许修改现有的继承关系，不许增加新的类，可以在现在代码基础上改动

使得作业代码能够完成本作业要求的测试

0x00 分析

本次作业目的为找出程序中的错误。

0x01 Test

末尾多了一个 }。

0x02 GameObject

  1. defaultStrength 未使用，应删除。
  2. 构造方法中应指明 this。
  3. attack 应首先判断是否死亡。
  4. attack 应在攻击范围小于攻击距离时停止。
  5. 不应该通过类实例访问静态成员。
  6. changeHealth 和 dead 均未使用，应删除。

0x03 GameBase

  1. 缺少构造方法。
  2. createGameBase 中需要传递参数。
  3. setHealth 参数类型应为 int，下面 l00 应为 100。
  4. return 应使用英文半角分号。
  5. building 缺少返回值。

0x04 Building

需要调用父类构造方法传递参数。

0x05 Barrack

  1. 需要调用父类构造方法传递参数，设置生命值。
  2. traing 缺少 new。

0x06 Soldier

Soldier 拼写错误。

0x07 RifleSoldier&RPGSoldier&Dog

RifleSoldier 和 RPGSoldier 的构造方法应调用父类无参数构造方法。

Dog 需修改 attack。

Thanks for your reading.
