新起点

发表于 2023-04-03 分类于 生活 阅读次数： 本文字数： 351 阅读时长 ≈ 1 分钟

新的样子，新的开始。

忙碌了 5+3 个月，很多重要的事情终于有了结果。过去的一个月可能是比较 challenging 的一个月，面对种种未知和阻碍，最后还是挺了过来。很难说收获了什么经验，只不过是随着时间的流逝而不得不面对罢了。

话说回来，从寒假开始构思如何改造本站的 UI 以更符合我的审美，最终做了如下更改：

  1. 不再使用英文为主要语言，改为完全中文站；
  2. 改用 霞鹜文楷 作为主要字体；
  3. 重构许多 UI 细节，增加圆角等；
  4. 重构色彩搭配，重新设计了亮暗模式；
  5. 不再使用 utterances 为评论系统，改为使用 giscus，也就是把评论系统从 Issues 改为 Discussions；
  6. 其他若干修改。

当然，这次对 NexT 主题的魔改也存在不合理之处，目前发现的 bug 有：

  1. 评论需要手动刷新才加载；
  2. 评论在暗色模式下仍呈现亮色；
  3. 字体的粗体格式很不明显
  4. 其他若干问题，希望你可以在下方评论区提出。

现在开始，是本站的新起点，也是我的新篇。

最近应该会更新几篇文章，敬请期待（在写了在写了）。
